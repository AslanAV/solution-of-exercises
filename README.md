## Setup
```sh
$ git clone https://github.com/AslanAV/solution-of-exercises

$ make install
```

## Run tests & linter
```sh
$ make test

$ make lint

$ make lint-fix
```

## Test Coverage
```sh
$ make test-coverage
```
