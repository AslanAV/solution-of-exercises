## Setup
```sh
$ git clone https://github.com/hexlet-boilerplates/php-package.git

$ make install
```

## Run tests & linter
```sh
$ make test

$ make lint

$ make lint-fix
```

## Test Coverage
```sh
$ make test-coverage
```
